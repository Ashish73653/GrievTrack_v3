# Security Policy

## Reporting vulnerabilities
- Email the maintainers or open a private security advisory on GitHub.
- Include steps to reproduce and any logs or payloads that demonstrate impact.

## Scope
- Flask app (`app.py`), ledger abstraction (`ledger_backend.py`), and Fabric stub.
- No production authentication is present; treat deployments as prototypes.

## Expectations
- We will acknowledge reports within 72 hours.
- Please avoid public disclosure until a fix is available.
