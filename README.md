# GrievTrack_v3

Minimal Flask scaffold for GrievTrack.

## Setup
1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Run
```bash
python app.py
```
Then open http://127.0.0.1:5000/ in your browser. The root path redirects to the dashboard and navigation links are available for all Phase 0 pages.
